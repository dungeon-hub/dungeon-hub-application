## Dungeon Hub Bot

This discord bot written in Kotlin uses the following libraries:
- [Kord Extensions](https://github.com/Kord-Extensions/kord-extensions) to connect to discord
- [Our hypixel wrapper](https://github.com/dungeon-hub/hypixel-wrapper/) to use the Hypixel API
- [The api package](https://github.com/dungeon-hub/dungeon-hub-api) to access the data stored in [our backend](https://github.com/dungeon-hub/dungeon-hub-server)

You can check out our commands in the [commands directory](src/main/kotlin/net/dungeonhub/application/commands)
